Ready to use instant sites built for Elementor Page Builder and Munk WordPress theme, Beaver Builder and the WordPress Gutenberg Editor. Import free designs with one click and get started with amazing website.

== Description ==

Munk Sites plugin will help you import the demo content, including settings of the widgets and the customizer, with a click.

The demo content will make your website look like the preview of a theme so that you get a basic guideline for making your website.

Once installed and activated, Munk Sites will be accessible through **Appearance > Munk Sites**.

== Installation ==
1. Upload `munk-sites.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Dashboard > Appereance >> Munk Sites to start importing ready to use designs.
